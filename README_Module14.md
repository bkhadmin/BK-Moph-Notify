# Module 14

สิ่งที่เพิ่ม
- ตัวอย่าง Flex Message สำหรับข้อมูลนัดหมายจาก query
- รองรับทั้งแบบ
  - single bubble
  - carousel
  - top 5 dashboard
- มีตัวอย่าง mapping field จริง:
  - clinic_name = คลินิก
  - department = แผนกที่นัด
  - total_appointment = จำนวนผู้ป่วยที่นัด
  - sent_at = วันที่และเวลาส่งข้อความ

แนะนำการใช้งาน
- ถ้าต้องการสวยและอ่านง่าย: ใช้ `appointment_single_bubble.json`
- ถ้าต้องการส่งหลายคลินิกในครั้งเดียว: ใช้ `appointment_summary_carousel.json`
- ถ้าต้องการสรุปผู้บริหาร: ใช้ `appointment_top5_dashboard.json`
